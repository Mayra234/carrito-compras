# Prueba Técnica - Carrito de Compras (Next.js + Redux)

Proyecto de ejemplo para la prueba técnica.

## Tecnologías

- Next.js
- React 19
- Redux Toolkit
- Bootstrap 5

## Instrucciones

1. `npm install` o `yarn`
2. `npm run dev` (abre http://localhost:3000)

## Notas

- Persistencia con localStorage en `src/utils/localStorageMiddleware.js`.
- Carrito flotante (drawer) en `src/components/CartDrawer.jsx`.
- Botón flotante está en la esquina inferior derecha y es responsive (CSS media queries).
- Las imágenes estan en `public/img/`.
